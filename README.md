🧪 FTIR Analysis for Scientific Research
تحليل FTIR للبحث العلمي

This repository provides a complete workflow for analyzing FTIR spectra related to osmotic dehydration and thermal drying of natural products such as ginger and potato.
يقدّم هذا المستودع بيئة تحليلية كاملة لتحليل أطياف FTIR المرتبطة بالمعالجة الأسموزية والتجفيف الحراري للمنتجات الطبيعية مثل الزنجبيل والبطاطس.
